# 專案管理 Prompt 範本

> 回到 [Prompt 範本集索引](./index.md) | [研究筆記](../google-workspace-gemini-prompt-guide.md)

## UAT 測試案例

```
Create a table with 10 user acceptance tests (UAT) for the website
registration form.
```
**工具**：Gemini App → 複製到 Sheets

### 迭代：交辦信

```
Draft an email to [my colleague] who is running this UAT and explain what
they need to do next.
```

## 會議紀錄與摘要

使用 **Gemini in Meet** 的 Take notes for me 功能。會後開啟生成的 Doc：

```
Summarize this call transcript in a short paragraph. In bullet points,
highlight the action items, decisions made, and owners for each item
based off of [call transcript].
```

## 專案狀態更新模板

```
Draft a project status update email template to send to my manager.
Include sections for a summary of key accomplishments this week, any
challenges faced, and the top three priorities for next week.
```

## 專案回顧

### 產出引導問題

```
I need to write a report detailing the successes, failures, and lessons
learned from [project]. Draft a list of 20 questions to guide a cross-team
process investigation. Include questions to uncover what worked, what
didn't, specific process breakdowns, technical issues, communication gaps,
or any other potential contributing factors to the problem or success of
the project.
```

### 摘要回顧報告

```
Summarize this document in two paragraphs. Include high-level information
about the project's goals, the main contributors, the outcomes of the
project, and any key successes or failures.
```

## Issue Tracker

```
Create a spreadsheet to track project issues, including descriptions,
status, assigned owner, and action items for resolution.
```
**工具**：Gemini in Sheets

### 問題通知信模板

```
Draft an email template to announce when an issue arises and include
causes, solutions, and timelines to resolve it.
```

### 升級通知信模板

```
Draft an email template to a stakeholder to escalate a critical project
issue, outlining the impact and proposed solution.
```

## Workback Schedule

```
I am a [technical project manager] at [company] overseeing [project and
brief project description]. The project has the following scope: [scope].
Our project goals are: [project goals]. Our project deliverables are:
[project deliverables]. Our budget is [budget], and our delivery date is
[delivery date]. Help me create a workback schedule to keep the team on
track. Include dates for key milestones and demos.
```
**工具**：Gemini App
