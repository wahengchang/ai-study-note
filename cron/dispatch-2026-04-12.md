## Execution Protocol

1. Read `.automation/issues.json` for current state

4. Execute the agent pipeline: `@writer` (draft) → `@diagram` (if assigned) → `@reviewer` (QA) → `@content-ops` (intake + classification)
5. Commit, push, open PR
6. Update tracker status to `pr_open`
