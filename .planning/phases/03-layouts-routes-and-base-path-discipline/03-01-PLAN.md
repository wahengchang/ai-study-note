    ```css
    @theme {
      --color-bg-global: #050505;

      --color-brand-orange: #FF4F00;

      --font-mono: "JetBrains Mono", "Fira Code", ui-monospace, monospace;
      --font-sans: Inter, Geist, system-ui, sans-serif;
    }

    html {
      background-color: var(--color-bg-global);
      color: var(--color-text-primary);
      font-family: var(--font-sans);
      font-size: 14px;
    }

    body {
      min-height: 100vh;
    }
    ```

    Do NOT add additional rules beyond the above. Per visual-guideline.md: no gradients, no shadows, no blur. The 14px base + sans body + mono headings are the whole language.
